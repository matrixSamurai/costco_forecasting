# Pre-Model 1: Severe Weather Classification

## Overview

Pre-Model is a binary classifier that sits **before Model 1** in the pipeline. It predicts whether severe weather events (fog, snow/ice, hail, thunderstorm) are likely based on weather station observations, and decides whether to run Model 1 or skip it.

```
Weather Data → Pre-Model (Will there be severe weather?)
                 ├─ No  → delay = 0, skip Model 1, go to Model 2
                 └─ Yes → Model 1 (compute delay %) → Model 2 (compute loss)
```

## Why Pre-Model Exists

Model 1 is a regression model trained on **synthetic delay data** (generated by a formula). Pre-Model is trained on **real weather event records** from 4.34 million NOAA weather station observations (2021-2025), providing an independent signal that Model 1 doesn't have.

## Training Data

**Source**: `data/source/weather_data_2021_2025.csv` (4.34M raw → 2.32M after cleaning)

**Features (10):**

| Feature | Description |
|---------|-------------|
| temp_mean | Mean temperature (°F) |
| temp_min | Min temperature (°F) |
| temp_max | Max temperature (°F) |
| prcp_total | Precipitation (mm/day) |
| snow_depth | Snow depth (cm) |
| visibility | Visibility (miles) |
| wind_speed_mean | Mean wind speed (mph) |
| wind_gust_max | Max wind gust (mph) |
| month | Month (1-12) |
| day_of_year | Day of year (1-365) |

**Label**: `has_severe_weather = 1 if fog OR snow_ice_pellets OR hail OR thunder`

**Distribution**: 22.4% positive / 77.6% negative

## Model Performance

| Model | Accuracy | Precision | Recall | F1 | ROC-AUC |
|-------|----------|-----------|--------|-----|---------|
| **XGBoost** | 82.4% | 57.6% | 81.9% | 67.6% | **90.5%** |
| Random Forest | 81.8% | 56.5% | 81.6% | 66.8% | 90.0% |
| LightGBM | 82.3% | 57.3% | 81.9% | 67.5% | 90.4% |

**Top features**: visibility (52.4%), temp_mean (12.0%), prcp_total (10.5%)

## API Endpoint

**POST /api/classify** — Standalone classification

```json
// Request
{"weather": {"temp_mean": 20, "visibility": 1, "snow_depth": 8, ...}, "date": "2024-01-15", "model": "all"}

// Response
{"classifications": {"XGBoost": {"has_delay": true, "probability": 0.9987}, ...}}
```

## Integration

- **Route delays** (`/api/route-delays`): Classifies every pitstop along the route. Flags severe if >20% pitstops affected or max probability >70%.
- **Route loss** (`/api/route-loss`): Skips Model 1 entirely when no severe weather detected.
- **Pipeline** (`pipeline.py`): Same skip logic in `predict_revenue_loss()`.
- **Frontend**: Classification banner shown on both main page and demo page.
